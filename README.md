# Reaction Time Test

A tiny reaction time game built with just HTML, CSS and vanilla JavaScript. 
Ideal for GitHub Pages and old devices – no frameworks or heavy assets.

## Play

Open `index.html` or enable GitHub Pages on this repository.

Click **Start**, wait for the box to turn green, then click/tap as quickly as you can. 
Your reaction time will be displayed in milliseconds. Hit **Try again** to play another round.
